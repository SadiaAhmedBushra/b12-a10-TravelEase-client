# TravelEase

Welcome to **TravelEase**, your ultimate platform for easy and reliable vehicle rentals. Whether you're planning a weekend getaway or a long road trip, TravelEase connects you with a wide range of vehicles tailored to your needs.

## Features

- **Secured Authentication**: Safe and reliable user authentication to protect your personal data.
- **Wide and Verified Vehicle Options**: Choose from wide range of vehicles that are verified for your safety and reliability.
- **User-Friendly Booking System**: Browse and book vehicles with aN intuitive interface.
- **Customer Reviews**: Check out customer feedback before you make a choice.
- **Vehicle Management by Registered Users**: Registered users can add their own vehicles, edit and delete them as well.
- **Real-time Updates**: Vehicles and bookings are updated dynamically without page reloads.
- **Responsive Design**: Responsive layout for on desktop, tablet, and mobile devices.


## Technologies Used

- **Frontend**:
  - React.js (with React Router)
  - Tailwind CSS
  - React Icons
  - Framer Motion
  - Swiper.js
  - React Toastify
  - SweetAlert2

- **Backend**:
  - Node.js with Express.js
  - MongoDB (Atlas)
